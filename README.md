# My Developed Configuration Files (Dotfiles)

Welcome to my personal configuration files repository! This collection of dotfiles helps me set up a consistent and efficient development environment across different machines.

I use these configurations for:
- Command-line productivity
- Text editing (Vim)
- Git version control
- Code style and formatting
- Personal customization and automation

---

## Contents

| File/Folder          | Purpose                                      |
|----------------------|----------------------------------------------|
| `.bashrc`            | Shell setup                                  |
| `.vimrc`             | Vim editor configuration                     |
| `.gitconfig`         | Git global settings                          |
| `README.md`          | This file                                    |
